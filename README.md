This repository contains the **Static HTML/CSS Version** of the "Hope in a Box" web application.

Because **GitHub Pages** does not support server-side languages like PHP or database connections (MySQL), this version serves as a **frontend design showcase**. It demonstrates the user interface, responsiveness, and layout of the project without the dynamic data processing.

## üöß Static Mode Limitations
* **No Database Connection:** Dynamic features (e.g., calculating donation totals, tracking live volunteer slots) are simulated or hard-coded.
* **Forms Disabled:** The "Contact Us", "Donate", and "Volunteer" forms do not submit data.
* **Static Buttons:** Clicking "View Details" or "Volunteer" buttons will trigger a pop-up alert explaining that the feature requires a live database connection.
